# 🥁 Drum Kit Web App

An interactive and fun **Drum Kit Web Application** that allows users to play different drum sounds using keyboard keys or mouse clicks.

## 🚀 Live Demo
👉 [Drum Kit](https://parthl001.github.io/Drum-Kit/)

---

## 📌 Features

- 🎵 Play drum sounds using keyboard keys
- 🖱️ Click on buttons to trigger sounds
- ⚡ Instant audio feedback
- 🎨 Clean and responsive UI
- 🔊 Realistic drum sound effects

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

---

## 🎮 How to Use

1. Open the live demo link  
2. Press the following keys on your keyboard:

   | Key | Sound        |
   |-----|-------------|
   | W   | Crash       |
   | A   | Kick Bass   |
   | S   | Snare       |
   | D   | Tom 1       |
   | J   | Tom 2       |
   | K   | Tom 3       |
   | L   | Tom 4       |

3. Or simply click on the drum buttons displayed on the screen.
